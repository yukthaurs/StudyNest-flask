A simple web application to manage study tasks, built with Python and Flask.

## Features
- Add study tasks dynamically
- View all tasks in a clean, modern interface
- Responsive and minimal design

## Technologies Used
- Python 3.14
- Flask
- HTML & CSS

## How to Run
1. Clone the repository:  
   `git clone https://github.com/yukthaurs/studynest-flask.git`
2. Navigate into folder:  
   `cd studynest-flask`
3. Install Flask:  
   `pip3 install flask`
4. Run the app:  
   `python3 main.py`
5. Open browser:  
   `http://127.0.0.1:5000/`
